# Photography Portfolio

## Technologies Used

- Vite | build tool
- Tailwind | mini styling
- React | frontend
- Node.js | runtime
- Shadcn UI | inspiration for UI component
- Vercel | hosting

## Features

- Displays a collection of my photography work
- Responsive design for mobile and desktop
